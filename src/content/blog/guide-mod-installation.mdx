こんにちは、管理人です。
「動画で見たあのMODで遊びたい！」と思っても、導入のハードルって意外と高いですよね。

この記事では、最も一般的な **Java版** でのMOD導入手順を解説します。

## 1. 前提MOD（ローダー）を入れる
MODを動かすための土台ソフトです。遊びたいMODに合わせて選びましょう。

* **Minecraft Forge**: 昔からの定番。大型MOD（黄昏の森など）はだいたいこれ。
* **Fabric**: 最近の人気者。動作が軽くてバグが少ない。
* **NeoForge**: 新しい規格。最新バージョンで増えています。

### 導入手順
1.  公式サイトから「Installer」をダウンロード。
2.  ダウンロードしたファイルをダブルクリックして実行。
3.  「Install Client」を選んでOKを押すだけ！

## 2. MODファイルをダウンロードする
怪しいサイトはNGです。以下の2大サイトから落とすのが安全です。
* **CurseForge**
* **Modrinth**

## 3. modsフォルダに入れる
ここが一番の難関ですが、ショートカットキーを使えば一発です。

1.  キーボードの `Winキー` + `R` を押す。
2.  出てきた画面に `%appdata%` と入力してエンター。
3.  `.minecraft` フォルダを開く。
4.  その中に `mods` というフォルダがあります（なければ右クリックで作る）。
5.  そこにダウンロードしたMODファイル（.jar）を放り込む！
